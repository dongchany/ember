# Legacy Archive

This directory stores files from the pre-Qwen3.5 phase.

Purpose:
- Keep historical reports, scripts, and long-form tutorial prompts.
- Reduce noise in the main source tree during the Qwen3.5 refactor.
- Preserve easy rollback and reference without deleting project history.

Current archived groups:
- `legacy/docs/`: old task checklists, tutorial prompts, and stage-specific docs.
- `legacy/scripts/`: old analysis/report/train/verifier utilities and comparison scripts.

Not archived intentionally:
- `sglang-env/`, `torch-env/`, `vllm-env/` remain at repository root for framework comparison tests.
